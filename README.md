## Сюжет

Ты — студент, который решил схитрить и **взломать БРС** (базу рейтинговой системы), чтобы поставить себе оценки. Проблема в том, что по ночам в корпусе бродит **Алгем** — и он явно не рад твоим планам.

С **0:00** до **6:00** тебе нужно:

- **Взломать сервер** через ноутбук — автоматический прогресс, но привлекает Алгема
- **Следить за камерами** через планшет — 7 комнат + 4 вент-камеры
- **Блокировать вентиляцию** — закрывай шахты, пока Алгем не добрался через венты
- **Включать звуковые приманки** — отвлекай Алгема звуком на камерах
- **Закрывать двери** — последний рубеж, когда Алгем уже в офисе

Если Алгем до тебя добрался — **game over**. Дожил до 6 утра и сервер взломан — **победа**.

## Управление

| Клавиша | Действие |
|---------|----------|
| `TAB` | Открыть / закрыть планшет |
| `←` `→` | Панорама камеры (обычные 1-7, венты 8-11 циклически между собой) |
| `1`-`9` | Переключиться на камеру по номеру |
| `WASD` / Стрелки | Оглядываться в офисе |
| `E` | Взаимодействовать с сервером |
| `L` | Открыть ноутбук |
| Клик **MAP** | Переключить камеру / карту вентов |
| Клик **BAIT** | Включить звуковую приманку на текущей камере |
| Клик **SEAL** | Заблокировать шахту вентиляции |
| `ESC` | Закрыть ноутбук / вернуться в меню |
| `F11` | Полноэкранный режим |

## Система ИИ

Алгем использует **конечный автомат** с тремя состояниями:

| Состояние | Поведение |
|-----------|-----------|
| **IDLE** | Размышляет, агрессия копится пассивно |
| **PATROL** | Случайное блуждание по камерам, обходит наблюдаемые |
| **ATTACK** | A* поиск пути напрямую к офису |

ИИ реагирует на действия игрока:
- Включённый сервер повышает шкалу внимания Алгема
- Реклама привлекает Алгема после 2-секундного окна безопасности
- Звуковые приманки перенаправляют Алгема к источнику звука (в радиусе 2 комнат)
- Долгое наблюдение за камерой увеличивает стоимость рёбер — Алгем обходит эту комнату
- Seal блокируют выходы из вентов, заставляя идти длинным путём

## Система вентиляции

Два вентиляционных канала (`VENT_A`, `VENT_B`):
- **4 точки блокировки** — клик для закрытия, 5 секунд на SEAL
- **Случайные поломки** — венты периодически ломаются, открывая короткие пути
- **Кнопки сброса** — восстанавливают сломанные венты
- При поломке вента Алгем получает **короткий маршрут** к офису
